Language Pack

Language Pack

打开应用商店，输入 Language Pack ，找到想要安装的语言插件进行安装，安装完成后点击右下角重启 Visual Studio Code ，即可切换到指定的语言。或者，在安装好指定的语言插件之后，使用快捷键 Ctrl + Shift + P 打开命令面板，然后输入 >Configure Display Language 即可配置显示语言。
